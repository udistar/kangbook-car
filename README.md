# 강북 페차장 홈페이지

## 프로젝트 개요
강북에 위치한 전문 폐차장의 홈페이지입니다. 20년 경력의 전문 폐차 업체로서 신뢰할 수 있는 폐차 서비스를 제공합니다.

## 주요 기능
- **회사 소개**: 20년 경력의 전문 폐차 업체 소개
- **폐차 안내**: 상세한 폐차 절차 및 비용 안내  
- **폐차종류**: 6가지 차량 종류별 폐차 정보 (일반차, 화물차, 특수차, 오토바이, 농기계, 건설기계)
- **서비스 안내**: 일반 폐차, 중고부품 판매, 무료 견인, 서류 대행 서비스
- **폐차 절차**: 4단계 폐차 절차 안내
- **무료 견적 문의**: 개선된 폼 디자인과 시각적 피드백
- **연락처 정보**: 업데이트된 사업자 정보
- **지도 서비스**: 네이버 맵과 카카오 맵으로 쉬운 찾아오기

## 🎨 디자인 특징

### 현대적 다크모드 스타일
- **다크 테마**: #0f172a, #1e293b, #334155 등 세련된 색상 조합
- **그라디언트 효과**: 주요 버튼과 타이틀에 블루 그라디언트 적용
- **Inter 폰트**: Google Fonts의 Inter 폰트로 깔끔한 가독성
- **카드 기반 레이아웃**: 모던한 그림자와 호버 효과

### 개선된 UI/UX
- **폼 가시성 향상**: 대비가 뚜렷한 폼 필드와 버튼
- **상태 피드백**: 제출 성공/실패 시 시각적 피드백
- **스크롤 애니메이션**: 부드러운 페이드인 효과
- **반응형 디자인**: 모든 기기에서 최적화된 표시

## 기술 스택
- HTML5
- CSS3 (반응형 디자인)
- JavaScript (ES6+)
- Font Awesome 아이콘

## 현재 구현된 기능

### 1. 메인 페이지 (index.html)
- 반응형 네비게이션 바
- 히어로 섹션 with CTA 버튼
- 회사 소개 및 통계
- 서비스 안내 카드
- 폐차 절차 단계별 안내
- **지도 서비스**: 네이버 맵과 카카오 맵 탭 전환
- 문의하기 폼
- 푸터 정보

### 2. 폐차 안내 페이지 (guide.html)
- 폐차 정의 및 필요성 설명
- 폐차가 필요한 4가지 경우
- 상세한 4단계 폐차 절차
- 폐차 비용 정보 (10만원 ~ 50만원)
- 자주 묻는 질문 (FAQ) with 토글 기능
- CTA (행동 유도) 섹션

### 3. 폐차종류 페이지 (car-types.html) - 새로 추가
- **6가지 차량 종류**: 일반 자동차, 화물차, 특수차량, 오토바이, 농기계, 건설기계
- **카테고리별 상세 정보**: 대상 차종, 폐차 특징, 필요 서류, 예상 비용
- **비교 표**: 난이도, 소요시간, 예상비용, 견인필요, 중고부품 가치 비교
- **인터랙티브 기능**: 카테고리 클릭시 상세 정보 표시
- **시각적 난이도 표시**: 쉬움(초록), 보통(주황), 어려움(빨강)

### 4. 스타일 (css/style.css, css/guide.css, css/car-types.css)
- 모바일 우선 반응형 디자인
- 현대적인 그라디언트 디자인
- 호버 효과 및 애니메이션
- 모바일 메뉴 토글
- 카드 기반 레이아웃
- FAQ 아코디언 스타일

### 5. JavaScript 기능 (js/main.js, js/guide.js, js/car-types.js)
- 모바일 메뉴 토글
- 스무스 스크롤링
- 폼 유효성 검사 및 제출
- 전화번호 자동 하이픈 추가
- 차량번호 형식화
- 스크롤 애니메이션
- FAQ 아코디언 기능
- **카테고리별 폐차 정보**: 인터랙티브한 차량 종류 선택 기능 (새로 추가)
- **비교 표**: 난이도 및 비용 비교 기능 (새로 추가)
- **지도 탭 전환**: 네이버 맵과 카카오 맵 간 전환

## 사용 방법
1. 웹 브라우저에서 `index.html` 파일을 열어 메인 페이지 확인
2. `guide.html` 파일을 열어 폐차 안내 페이지 확인 
3. `car-types.html` 파일을 열어 폐차종류 페이지 확인 (새로 추가)
4. 네이버 지도 또는 카카오 지도로 찾아오는 길 확인
5. 모바일에서는 반응형으로 자동 조절됨
6. 견적 문의는 폼을 통해 신청 가능
